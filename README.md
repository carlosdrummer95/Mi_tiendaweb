# Mi Tienda Web

Aplicación web para la venta de computadores y aparatos electrónicos, 
desarrollada en **Java**, usando **Servlets, JSP** y conexión a **MySQL**.

---

## 🚀 Requisitos
- **JDK 17**
- **NetBeans 21**
- **Apache Tomcat 9**
- **MySQL** (XAMPP o Workbench)
